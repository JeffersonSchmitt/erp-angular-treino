# erp-angular-treino

Descrição
=========


Aplicação web visualmente conta com:

- Dashboards com gráficos
- Formulários de cadastros
- Relatórios
- Grids de exibição

Regras de negócio
=========
O usuário deve conseguir:
- Cadastrar
- Visualizar
- Controlar estoque
- Controlar vendas

======================
